Task 2.6: Some Programming on the WebServer

In this task we implement code for some api Get requests as listed below in the sub-tasks.
The api calls can be made as per instructions provided under each request.

  • Sub-task 2.6.1 Multiply:
    This Get request takes TWO integer argumemnts and returns the result of the multiplication of the arguments.
    This api request is made by adding the query to the host name and supply the required arguments as shown below.

    /multiply?num1=(value of the first argument)&num2=(value of the second argument).

    For example: /multiply?num1=8&num2=5

  • Sub-task 2.6.2 GitHub:
    This Get request query GitHub repository and returns the repo name, ID and login information of the owner of the repo.
    To make this api call, follow the instruction below:

    hostname/github?query=users/repoName/repos

    For example: /github?query=users/amehlhase316/repos

  • Sub-task 2.6.3 Make your own requests
    1. BMI Calculator: Calculates the BMI and returns the result
      This api Get request is made as demonstrated below.
      /bmi?weight=value of body weight in kg&height=value of height in meter).

      For example: /bmi?weight=80.0&height=2.0

    2. Maximum of Two Numbers: Comapares two numbers and returns the larger number
      This api Get request is made as shown below.
      /max?num1=value of the first number&num2=value of the second number

      For example: /num1=250&num2=80
